# PDF Inspection Form Automation

An ASP.NET Core MVC application that digitizes inspection forms by collecting user input through a web interface and automatically writing the data onto a predefined PDF template using iText7.

## Technologies
- ASP.NET Core MVC
- C#
- Bootstrap
- iText7

## Features
- Dynamic PDF generation
- PDF template editing
- Form-based data input
- PDF preview system
- Inspection form automation
